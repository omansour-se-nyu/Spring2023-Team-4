# Spring2023-Team-4
This is team repository for Team 4 (Spring 2023)

## Quickstart
1. Install all the packages:
    ```shell
    pipenv install
    ```
2. Start the server:
    ```shell
    pipenv run python server/main.py
    ```
3. Start the client:
    ```shell
    pipenv run python client/main.py
    ```
